# Java Swing Image Compression App

## Compile and Run

1. Open terminal in `c:/Imagecompression`
2. Compile: `javac *.java`
3. Run: `java ImageCompressorGUI`

## Features
- Select JPG/PNG images
- Adjust compression quality slider (0-100)
- Preview original and compressed side-by-side
- See file size reduction and compression %
- Save compressed JPEG

## Notes
- Uses built-in Java libraries only (ImageIO, Swing)
- Compression uses JPEG quality parameter for size reduction
- Previews scale large images for performance
